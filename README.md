#  Interactive Generative Plotting with AxiDraw

## Hardware
- Macbook Pro
- AxiDraw v3  
- Arduino metro mini  
- 2 x Ultrasonic sensor: [HC-SR04](https://www.electroschematics.com/wp-content/uploads/2013/07/HCSR04-datasheet-version-1.pdf)
